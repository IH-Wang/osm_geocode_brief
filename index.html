OpenStreetMap Nominatim Service

Nominatim 是一個基於 osm 開發的搜尋引擎， 它主要功能是將地址轉經緯度或經緯度轉地址。

它提供兩種方式給一般 osm 用戶作使用。

  • 輕量級用戶建議直接使用它們的 api 做使用就行
  • 頻繁的應用則推薦自己架設 Nominatim 的 server，官網也有提供環境建置教學流程，網路上也有包好的docker image可以直接使用。

基本安裝

  • 環境: Linux Ubuntu 18.04, 20.04 | CentOS 7.2, 8

  • 軟體需求:

    • 編譯用:

      cmake, expat, proj, bzip2, Zlib, Boost libraries(including system and filesystem), PostgreSQL client libraries, a recent C++ compiler (gcc 5+ or Clang 3.8+)

    • 執行用:

      PostgreSQL (9.3+), PostGIS (2.2+), Python3, Psycopg2, PHP (7.0 or later), PHP-pgsql,PHP-intl (bundled with PHP), a webserver (apache or nginx are recommended)

基本安裝

  • III. 硬體需求:

    • 記憶體: 最低2GB否則會建置失敗，如果要跑整個地球的地址搜尋，則需要至少64GB。

    • 硬碟空間: 至少800GB，需要將所需的osm圖資匯入資料庫，而osm圖資也日漸增長，另外使用SSD硬碟則可加快匯入圖資速度及查詢速度。

HERE STUDIO

HERE Studio/Data Hub 是 HERE Technologies 所推出的一個線上地圖製作平台，其功能有兩大核心 Data Hub，以及 Studio。

主要特色

  • 強大的地圖資料管理和顯示工具

  • 有限的免費且方便的的雲端服務

  • 嚴謹的權限管理功能

  • 快速方便的 UI 元件

  • 快速的處理調整並發布

地圖相關服務 API

  • HERE Data Hub / Studio：地圖資料儲存、發布與地圖製作

  • HERE Map Tile API：點陣圖（PNG/JPEG）地圖磚

  • HERE Geocode and Search API：經緯度與地址查詢、反查

  • HERE Waypoints Sequence：路徑點最佳化排序

  • HERE Routing API：路徑計算與詳細指引

  • HERE Route Matching：接收gps軌跡檔分析並回傳結果

  • HERE Matrix Routing：多點之間的空間關係計算

  • HERE Isoline Routing：特定時間或空間的可及範圍計算

  • HERE Tour Planning：針對物流業的使用場景而推出的一個新服務

整合應用服務 - 服務開始前

  • 車隊成本計算 - HERE Tour Planning
  • 物流路線規劃 - HERE Tour Planning / HERE Waypoints Sequence
  • 地理資料儲存 - HERE Data Hub / HERE Custom Location

整合應用服務 - 使用者端

  • 查詢地址或地點 - HERE Geocoder API / HERE Places API
  • 以時間或距離計算物流費用 - HERE Routing API
  • 即時動態地圖 - HERE JavaScript API / HERE SDK

整合應用服務 - 服務進行中

  • 管理中心的即時車隊動態地圖與訂單地圖 - HERE JavaScript API / HERE Data Hub / HERE Custom Location
  • 管理中心監控車輛或貨品的位置 - HERE Tracking
  • 管理中心監控車輛的目的地資訊，甚至動態更新每個目的地的抵達時間 - HERE Waypoints Sequence
  • 透過路徑計算的功能來得知抵達下一個地點的時間與距離，藉此通報客戶端以及管理中心，雙方都可以得知即時的動態與到達時間 - HERE Routing API
  • 當物流車輛接近客戶位置時，觸發事件告知客戶端準備進行收件或取件 - HERE Geofencing
  • 管理中心監控車輛是否有不正常的偏離路線的狀況，特別是車輛載送高價值貨物時 - HERE Geofencing
  • 針對臨時的急件，派出最接近的車輛前往服務 - HERE Matrix Routing
  • 針對特定的道路進行迴避，或是在客戶的私人場域中進行路徑規劃 - HERE Custom Routes
  • 司機端的語音導航、測速照相警告、管理中心的派遣功能 - HERE SDK Premium

整合應用服務 - 服務結束後

針對司機的駕駛行為進行分析，並採取必要措施降低風險 - HERE Route Matching
